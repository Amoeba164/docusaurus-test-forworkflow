# Изображения для сайта

## Что добавить сюда:

### Обязательные файлы:

1. **favicon.ico** (32x32px)
   - Иконка сайта в браузере
   
2. **logo.svg** 
   - Логотип для навигации
   - Желательно SVG для масштабируемости

3. **logo-dark.svg** (опционально)
   - Логотип для тёмной темы

### Опциональные файлы:

4. **docusaurus-social-card.jpg** (1200x630px)
   - OpenGraph изображение для соцсетей
   - Показывается при шаре ссылки

### Скриншоты и диаграммы:

Создайте подпапки:
- `screenshots/` - скриншоты интерфейса
- `diagrams/` - диаграммы и схемы
- `tutorials/` - изображения для туториалов

## Как использовать:

В Markdown файлах:

\`\`\`markdown
![Описание](/img/screenshot.png)
\`\`\`

В React компонентах:

\`\`\`jsx
import logo from '@site/static/img/logo.svg';
\`\`\`
